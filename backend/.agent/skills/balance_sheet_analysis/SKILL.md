# 资产负债表分析方法（无脚本版）

## 何时调用

当用户出现以下请求时调用本 skill：

- “分析资产负债表”“看资产质量”“看偿债能力”
- “解读某公司半年报/年报中的资产负债结构”
- “判断企业是否有流动性风险、应收风险、存货风险、在建工程风险、商誉风险”

若用户只要求“提取 PDF 内容”，应优先调用提取 skill；在有可读内容后再用本 skill做分析。

## 输入要求

- 优先输入：资产负债表相关结构化字段（如货币资金、短期负债、应收、存货、总资产、总负债、商誉、在建工程等）。
- 备选输入：财报文本片段（可来自 PDF 提取结果）。
- 当关键字段缺失时，必须在结论中明确“缺失字段及影响”。

## 输出约束

- 必须直接给出分析结论，不要要求用户“先确认后再继续”。
- 结论需包含：总体判断、主要风险点、支持证据、缺失信息与后续建议。
- 只基于已提供数据分析，不得编造具体数值。
- 不提供投资建议性结论（如“建议买入/卖出”）；仅做财务分析。

## 固定分析框架

### 1) 总览结构

- 先按五大块识别：流动资产、非流动资产、流动负债、非流动负债、所有者权益。
- 优先关注资产端真实性与可变现性；负债端通常相对更难作假但要看期限压力。

### 2) 流动性与短债压力（最高优先级）

- 重点看：货币资金 vs 短期有息负债/一年内到期负债。
- 规则：
  - 若货币资金明显小于短期负债，判定“短期偿债压力高/流动性风险上升”。
  - 若货币资金充裕但同时维持高息负债，提示“资金真实性、受限性或占用风险”，需多期跟踪。

### 3) 应收质量

- 重点看：应收账款、应收票据（区分银行承兑与商业承兑）、其他应收款。
- 规则：
  - 应收增长显著快于收入或行业常态时，提示回款与坏账风险。
  - 其他应收款占比偏高，提示潜在资金占用或利润调节风险。

### 4) 存货质量

- 重点看：存货规模、周转与跌价准备逻辑。
- 规则：
  - 普通行业存货久期越长，跌价风险越高；
  - 若长期不计提或计提明显偏弱，提示利润高估风险。
  - 对特殊行业（如高端白酒基酒）允许给出“行业例外说明”。

### 5) 非流动资产风险

- 固定资产：关注折旧/减值政策是否异常（过慢折旧可能美化利润）。
- 在建工程：规模大且长期不转固，提示资本化与资金去向风险。
- 无形资产/研发资本化：提示“费用化 vs 资本化”对利润的影响与操纵空间。
- 商誉：不摊销但需减值测试，关注未来一次性减值冲击利润的风险。
- 长期待摊费用：金额越小越稳健，异常偏大需警惕费用递延。
- 递延所得税：解释税会差异，不单独下强结论，需结合利润与现金流。
- 长期应收款：区分销售分期与融资租赁，前者回款风险更高。

### 6) 结论模板（必须按此顺序）

按以下结构输出：

1. `总体结论`：一句话概括资产负债表健康度。
2. `核心发现`：3-5 条，按风险等级排序（高/中/低）。
3. `证据与依据`：对应到具体科目与可见数据/文本证据。
4. `缺失信息`：列出无法判断的关键字段。
5. `后续跟踪建议`：给出可执行的财报跟踪点（多期对比、计提政策、回款变化等）。

## 语气与风格约束

- 使用审慎、可审计的措辞：如“提示”“可能”“需结合多期验证”。
- 避免绝对化断言：如“必然造假”“一定有问题”。
- 在数据不足时，不扩大结论范围。
